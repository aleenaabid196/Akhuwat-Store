# Akhuwat-Store
There are many NGOs that are supporting the under-privileged. Our task is to make a web based Ecommerce site for the merchandise of Akhuwat foundation. Sellers allowed by the foundation would be able to list the merchandise on the site and the user can purchase anything from the site. There should be a proper tracking system and implementation of payment API.

##Functional Requirements are as follow:

#### 1.Sellers should be able to register on the site. 
#### 2.NGO should be able to list all the applicant sellers and approve them to sell on the site.
#### 3.Sellers should be able to post product ads on the site and change the description, price etc.
#### 4.There should be a proper dashboard for the NGO to monitor the sales of all the sellers. And they can also change the commission rates for the sales.
#### 5.There should be a dashboard for the sellers also, to monitor their own sales, profits, top performing products etc.
#### 6.User should be able to go through the site to look for a product or he can also search for a specific product.
#### 7.Users can sort the product according to price tags, increasing price , decreasing price and best rated products.
#### 8.There should be around five search tags in each product listing using which the product would be listed in the searches.
#### 9.There should be categorization of products like clothes, household items etc.
#### 10.Users can add products to their favourites to order in the future.
#### 11.Users should be able to sign up using google sign in, through facebook and even email sign in. In case of email sign in, there should be an option to change the password.
#### 12.User should be able to see his order history.
#### 13.Users can rate the products after buying and sellers rating can be calculated using the ratings of all of the sales of all his products.
#### 14.Users can post questions related to a specific product that only the seller can answer (as in daraz.pk). And other users can also see the answers related to any specific product.
#### 15.There should be a proper tracking system for the products.
#### 16.Local payment APIs should be implemented in the project like JazzCash or any other that you can find.
#### 17.NGO should be able to foresee any seller account.
#### 18.It should allow the NGO to opt for only PrePaid orders and they can also turn on the cash on delivery option.
#### 19.There should be proper procedure to return the product in case of any error.
#### 20.There should be a chatbot frontend implemented (Just frontend). You do not need to use the NLP model or anything else, it's a project for the future.
